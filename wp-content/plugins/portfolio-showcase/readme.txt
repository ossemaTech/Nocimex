Easy to Use project/portfolio showcase management plugin for WordPress

== Description ==

Portfolio Showcase is a simple and easy to use  project/portfolio showcase management plugin. Portfolio Showcase uses custom post type for portfolio item. So adding new portfolio item is as simple as creating a new post. Add portfolio item in different categories and show then just using a simple short-code.

= Features =
* Portfolio Category Support
* Category Filter Support using Mixitup
* Modal Preview for Portfolio Details
* Use simple short-code [wpeden_portfolio] to embed portfolio
* Set total item in shortcode using items parameter (optional), [wpeden_portfolio items=3] 
* Set item per row in shortcode using ipr parameter (optional), [wpeden_portfolio ipr=3] 
* Translation Ready
 
== Installation ==


1. Upload `portfolio-manager` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
